Admin / AI

POST /admin-panel с полем section:

  • max — токен, шаблоны, long-poll
  • ai — Yandex AI, время report_time_msk=HH:MM
  • ai_report_now — тестовая отправка (не блокирует расписание)
  • ai_report_reset_today — снова разрешить автоотправку сегодня

Public: Max Webhook

GET/POST /max/webhook — без session, CSRF exempt.

{
  "updates": [
    {
      "update_type": "message_created",
      "message": {
        "sender": { "user_id": 7344745 },
        "body": { "text": "+79991234567" }
      }
    }
  ]
}
Ответ: 200 ok или 400 bad json. Связывает клиента по телефону → max_user_id.
